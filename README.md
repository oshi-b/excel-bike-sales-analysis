# excel-bike-sales-analysis
Data cleaning, analysis, and interactive dashboard creation using Excel on bike sales dataset.
# Bike Sales Data Analysis (Excel)
This project focuses on analyzing bike sales data to identify customer purchasing patterns and build an interactive dashboard using Microsoft Excel.

# Objective
To understand factors influencing bike purchases and provide insights based on customer demographics.

# Tools Used
Microsoft Excel
Pivot Tables
Charts & Slicers
# Dataset
The dataset contains:
Customer demographics (age, gender, marital status)
Income details
Occupation & education
Bike purchase status

# Data Cleaning Steps
Removed duplicate records
Standardized categorical values (Male/Female, Married/Single)
Formatted income and numerical columns
Created new columns for better analysis

# Dashboard Features
Bike purchase analysis by income
Customer segmentation (gender, marital status)
Purchase trends based on age group
Interactive filters using slicers

# Key Insights
Higher income customers are more likely to purchase bikes
Middle-aged customers show higher purchase rates
Married individuals tend to purchase more compared to single customers
Certain occupations have higher buying patterns
